# Demonstration 1: Plugin Structure and Installation

## What we will learn

- how to deploy and update a project level plugin
- the structure of a project level plugin
- basic QML

## What does it do?

This is a basic Hello World plugin.  
- It opens via a click on the camera icon to the right of the screen.  
- It displays a vanilla rectangle with a green title text.
- It is installed on the project level.  You will only see it when its project is open.

![alt text](img/demo1_screen.png)

- It also prints messages to the message log when it is opened and when it is closed. 

![alt text](img/demo1_log.png)
